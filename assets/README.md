# Assets

Generated marketing assets live here.

- `benchmark-chart-v2.png` is generated by `scripts/generate_benchmark_chart.py`.
- `reasoning-demo.png` is generated by `marketing/tools/generate_reasoning_demo.py`.
- `demo-comparison.mp4` is generated by `marketing/tools/generate_demo_video.py`.
- `marketing/assets/captures/*.json` are actual model outputs used by the video renderer.

Regenerate the chart:

```bash
python scripts/generate_benchmark_chart.py
```

Regenerate the video:

```bash
python marketing/tools/generate_demo_video.py
```

Audit raw model outputs:

```bash
python marketing/tools/capture_demo_outputs.py --runtime video
```

Regenerate the reasoning image:

```bash
python marketing/tools/generate_reasoning_demo.py
```
